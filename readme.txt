=== Plugin Name ===WP-SE Last PollsContributors: JooMoodDonate link: http://sociallife.it/blog/Tags: wordpress, social engine, integrationRequires at least: 2.7.1Tested up to: 2.7.1Stable tag: trunkThis plugin will display the Last X Social Engine Polls on your Wordpress blog.== Description ==This plugin/widget will display the Last X Social Engine Polls on your Wordpress blog.Note: This Plugin requires the Official Social Engine Polls Plugin: http://www.socialengine.net/order.phpPHP Requirements: PHP 5.x (not tested with PHP 4)Please, read the Installation Instructions in the Installation section of this page.You may customize a lot of features, such as:- the total number of polls- the total number of polls per-line- the date format- a lot of style preferences like: colours, borders, pic dimensions a.s.o.The widget and the plugin have separate configurations.Feel free to <a href='http://sociallife.it/blog/?page_id=58' target='_blank'>leave your comment here!</a>== Installation ==This section describes how to install the JooMood WP-SE plugin and get it working.1. First of all Unzip the file named `giggi_functions.zip`2. If you already uploaded a previous `giggi_functions` folder (for another JooMood plugin) skip the Steps #3, #4, #5 and go directly to the Step #63. Open the file named `giggi_database.php` inside the folder `giggi_functions`4. Fill the variables to connect to your Social Engine db; provide a path to your Social Engine Directory and to your Wordpress blog.5. Save and close the `giggi_database.php` file.6. Upload the 2 folders named `wp-se_polls` and `giggi_functions` to the `/wp-content/plugins/` directory. If you already uploaded `giggi_functions` folder (for another JooMood plugin) you don't need to upload it again.    7. CHMOD to 755 the file named `image.php` and the folder named `imagecache` (inside the `wp-se_polls` folder)8. Activate the plugin/widget through the 'Plugins' menu in WordPress9. Place `<?php joomood_polls(); ?>` in your templates== Frequently Asked Questions === Can I choose between usernames/full names? =Yes, you may choose to display your users by their username, or full names, or to hide both if you want.= May I change the tabs colours in order to have them like my blog ones? =Of course, you may customize the colours of every part of every box= May I change the dimensions of the pics? =Pics?
